# Module 17: Capstone Project

A comprehensive final project that integrates all skills from the course into a professional visualisation portfolio.

---

## Learning Objectives

- Design and implement a complete data visualisation pipeline
- Integrate multiple data sources and visualisation types
- Produce publication-ready figures and reports
- Present findings in a clear, compelling format

---

## Dataset

Use the CHIRPS rainfall dataset for Ethiopia (1981–2022) available in the datasets/ directory.

---

## Requirements

### Technical Requirements

- Python 3.10+ with matplotlib, numpy, xarray, cartopy, pandas
- Object-oriented Matplotlib API throughout
- Minimum 4 multi-panel figures
- At least one figure with a map projection

### Figures

1. **Climate Summary Dashboard** — 2x2 figure showing annual cycle, trend, distribution, and map
2. **Spatial Analysis** — Map series showing seasonal rainfall patterns
3. **Temporal Analysis** — Time series with anomalies, rolling statistics, and trend
4. **Comparative Analysis** — Multi-location comparison using shared axes

---

## Deliverables

1. Complete Python script(s) in scripts/
2. All output figures in images/
3. A Markdown report summarising findings
4. A Jupyter notebook version of the analysis
