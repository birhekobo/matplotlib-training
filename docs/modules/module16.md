# Module 16: Real-World Projects

Apply all skills learned to build comprehensive visualisation projects using real-world CHIRPS rainfall data.

---

## Learning Objectives

- Design complete visualisation workflows
- Build a rainfall analysis dashboard
- Perform anomaly analysis and visualisation
- Create a final portfolio of figures
- Document and present visualisation results

---

## Project 1: Rainfall Dashboard

Create a 2x2 dashboard with:
- Global precipitation map (Robinson projection)
- Regional East Africa zoom
- Time series of zonal mean
- Precipitation anomaly histogram

---

## Project 2: Time Series Analysis

Analyse precipitation anomalies:
- Compute monthly climatology
- Calculate anomalies relative to baseline
- Visualise with colour-coded bars
- Overlay trend lines

---

## Project 3: Comparative Analysis

Compare rainfall patterns across Ethiopian cities:
- Side-by-side climatologies
- Correlation analysis
- Seasonal breakdown
- Trend comparison

---

## Deliverables

- Python scripts for each project
- Publication-quality figures (PNG + PDF)
- Brief report explaining methodology and findings
