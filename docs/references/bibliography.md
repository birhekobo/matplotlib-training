## Core Matplotlib

### Paper

- **Hunter, J. D.** (2007). Matplotlib: A 2D Graphics Environment. *Computing in Science & Engineering*, 9(3), 90–95. [DOI: 10.1109/MCSE.2007.55](https://doi.org/10.1109/MCSE.2007.55)

The foundational paper describing Matplotlib's design, architecture, and capabilities.

### Documentation

- [Matplotlib Documentation](https://matplotlib.org/stable/) — Official user guide, tutorials, gallery, and API reference
- [Matplotlib Gallery](https://matplotlib.org/stable/gallery/index.html) — Thumbnail gallery with 300+ examples
- [Matplotlib Cheatsheets](https://matplotlib.org/cheatsheets/) — Official cheat sheets

### Books

- **Tosi, S.** (2020). *Matplotlib for Python Developers*. Packt Publishing.
- **McGreggor, D.** (2022). *Mastering Matplotlib*. Packt Publishing.

## Data Visualization

### Books

- **Tufte, E. R.** (2001). *The Visual Display of Quantitative Information* (2nd ed.). Graphics Press.
  - Classic text on statistical graphics and data-ink ratio.

- **Tufte, E. R.** (1990). *Envisioning Information*. Graphics Press.
  - Principles for displaying complex information.

- **Tufte, E. R.** (1997). *Visual Explanations: Images and Quantities, Evidence and Narrative*. Graphics Press.
  - Visual evidence and narrative design.

- **Cairo, A.** (2012). *The Functional Art: An Introduction to Information Graphics and Visualization*. New Riders.
  - Practical guide combining theory and practice.

- **Cairo, A.** (2016). *The Truthful Art: Data, Charts, and Maps for Communication*. New Riders.
  - Data visualization for communication and journalism.

- **Wilke, C. O.** (2019). *Fundamentals of Data Visualization*. O'Reilly Media.
  - Modern, practical guide to visualization design: [online version](https://clauswilke.com/dataviz/)

- **Wong, D. M.** (2010). *The Wall Street Journal Guide to Information Graphics*. W. W. Norton & Company.
  - Practical guide from a leading news graphics department.

- **Few, S.** (2009). *Now You See It: Simple Visualization Techniques for Quantitative Analysis*. Analytics Press.
  - Visual analysis and dashboard design.

### Papers

- **Ware, C.** (2012). *Information Visualization: Perception for Design* (3rd ed.). Morgan Kaufmann.
  - Comprehensive treatment of visual perception for design.

- **Heer, J., Bostock, M., & Ogievetsky, V.** (2010). A Tour Through the Visualization Zoo. *Communications of the ACM*, 53(6), 59–67.
  - Survey of visualization techniques.

## Citation Format

For academic use, please cite the course materials as:

> Matplotlib Training Course. (2024). *From Zero to Publication-Quality Visualizations*. [Online course materials].

And the key software:

```bibtex
@article{Hunter2007,
  author  = {Hunter, J. D.},
  title   = {Matplotlib: A 2D Graphics Environment},
  journal = {Computing in Science \& Engineering},
  year    = {2007},
  volume  = {9},
  number  = {3},
  pages   = {90--95},
  doi     = {10.1109/MCSE.2007.55}
}
```
