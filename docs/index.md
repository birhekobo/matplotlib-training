## Course Overview

This course teaches you to create **publication-quality visualizations** using Matplotlib, the most widely used plotting library for Python. You will progress from basic plots to advanced geographic visualizations using real-world satellite rainfall data from the **CHIRPS dataset**.

### What You Will Learn

- Core concepts of effective data visualization
- Matplotlib architecture: figures, axes, and artists
- Customizing plots for publication (fonts, sizes, color palettes)
- Working with subplots, insets, and complex layouts
- Statistical, scientific, and time-series visualization
- Geographic mapping with Cartopy and CHIRPS data
- Integrating with NumPy and Pandas
- Best practices for accessible, reproducible visualizations

### Prerequisites

- Basic Python programming knowledge
- Familiarity with NumPy arrays is helpful but not required

### Course Format

The course is structured as a **Jupyter Notebook** based training with:

- 17 modules with hands-on coding exercises
- Weekly assignments and quizzes
- A capstone project: rainfall visualization dashboard
- Self-paced learning with instructor support
